This is an KIEM (KIELER Execution Manager) data observer and producer
example that implements a table to display all variables/signals
during an execution.
To use this you also need the de.cau.cs.kieler.sim.kiem plug-in.

The W.T.K. (Sun Java Wireless Toolkit) must be installed
http://java.sun.com/products/sjwtoolkit/download.html
and additionally the 
Java Platform Micro Edition Software Development Kit 3.0
http://java.sun.com/javame/downloads/index.jsp
to compile this Java ME program.
