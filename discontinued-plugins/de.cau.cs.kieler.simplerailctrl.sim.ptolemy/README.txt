This plug-in transforms a SimpleRailCtrl model into an I/O equivalent
Ptolemy model (www.ptolemy.org). It uses the Eclipse Xtend language.

You need to provide the de.cau.cs.kieler.simplerailctrl plug-in and
also generate the model and edit code out of the /model/moml.genmodel for
the MoML file generation.
